HTML5

A linguagem HTML(sigla de Hyper Text Markup Language - Linguagem de Marcação de Hipertexto) é responsável pela criação de documentos para a web, as chamadas páginas. Ela são divulgadas na WWW(World Wide Web)- Teia de Alcance Mundial.

Por meio dessa linguangem, você pode especificar atributos para um texto(como fonte, tamanho, cor, etc.), e fazer o Hyperlinks que permitem navegar pela web. Para isso, ela utiliza marcações, chamadas tags, no ponto da página em que é feito uma ligação com outra.

Portanto, o HTML é um conjunto de tags utilizadas para edição de uma página de Internet. Tudo isso, feito através de uma linguagem fácil e simples de se trabalhar.
